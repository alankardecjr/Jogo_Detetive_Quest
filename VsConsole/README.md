# Versão Console - Detetive Quest

Esta é a versão em terminal do projeto, focada na lógica de investigação e no uso de estruturas de dados em C.

## Objetivo

O jogador explora uma mansão, coleta pistas, organiza evidências e tenta identificar o culpado com base em uma lógica de acusação.

## Conceitos aplicados

- Árvore binária de busca para o mapa da mansão e para o inventário de pistas;
- tabela hash para associar pistas a suspeitos;
- manipulação de memória e controle de fluxo em C.

## Como executar

1. Entre na pasta VsConsole.
2. Compile o arquivo principal com GCC.
3. Execute o programa gerado no terminal.

## Estrutura principal

- main.c: implementação completa da lógica do jogo;
- README.md: documentação da versão terminal.

## Destaque para recrutadores

Esta versão demonstra raciocínio algorítmico, compreensão de estruturas de dados e capacidade de transformar conceitos em uma aplicação funcional.
