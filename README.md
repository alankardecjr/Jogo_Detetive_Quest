# Detetive Quest

Projeto em C de um jogo de investigacao interativo que aplica estruturas de dados como arvores binarias e tabela hash para criar uma experiencia dinamica de exploracao, coleta de pistas e acusacao final.

Este repositorio reune duas versoes do projeto:

- versao console, com foco na logica e no uso de algoritmos e estruturas;
- versao grafica, com interface construida com Raylib.

## O que o projeto demonstra

- implementacao de arvore binaria de busca para organizacao e navegacao;
- uso de tabela hash para associar pistas a suspeitos;
- logica de gameplay com investigacao, coleta de evidencias e acusacao final;
- manipulacao de memoria em C;
- estrutura de projeto organizada para apresentacao e estudo.

## Estrutura do repositorio


## Tecnologias

- Linguagem C
- Raylib (versao grafica)
- GCC / MinGW
- Visual Studio Code

## Como executar

### Versao console

1. Entre na pasta VsConsole.
2. Compile o arquivo principal com GCC.
3. Execute o programa gerado.

### Versao grafica

1. Entre na pasta VsGrafico.
2. Certifique-se de que a Raylib esteja configurada corretamente.
3. Compile e execute o projeto.
