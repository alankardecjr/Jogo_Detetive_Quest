# Detetive Quest

Projeto em C desenvolvido como uma aplicação de investigação interativa com foco em estruturas de dados e lógica de jogo. O repositório reúne duas versões do projeto:

- uma versão em console, com foco na lógica e no uso de algoritmos e estruturas;
- uma versão gráfica, com interface construída com Raylib.

## O que o projeto demonstra

- implementação de árvore binária de busca para organização e navegação;
- uso de tabela hash para associar pistas a suspeitos;
- lógica de gameplay com investigação, coleta de evidências e acusação final;
- manipulação de memória em C;
- estrutura de projeto organizada para apresentação e estudo.

## Estrutura do repositório

```text
Detetive_Quest/
├── VsConsole/         # Versão terminal do jogo
├── VsGrafico/        # Versão com interface gráfica
├── README.md         # Visão geral do projeto
└── .gitignore        # Arquivos gerados pelo build
```

## Tecnologias

- Linguagem C
- Raylib (versão gráfica)
- GCC / MinGW
- Visual Studio Code

## Como executar

### Versão console

1. Entre na pasta VsConsole.
2. Compile o arquivo principal com GCC.
3. Execute o programa gerado.

### Versão gráfica

1. Entre na pasta VsGrafico.
2. Certifique-se de que a Raylib esteja configurada corretamente.
3. Compile e execute o projeto.
