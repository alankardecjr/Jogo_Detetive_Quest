# Versão Gráfica - Detetive Quest

Esta é a versão com interface visual do projeto, desenvolvida em C com a biblioteca Raylib.

## O que a versão oferece

- navegação entre cômodos por interface interativa;
- coleta automática de pistas;
- visualização das pistas coletadas;
- sistema de julgamento com validação lógica.

## Conceitos aplicados

- árvore binária para organização do mapa e das pistas;
- tabela hash para associação entre evidências e suspeitos;
- interface de usuário em C com Raylib;
- gerenciamento de estado do jogo.

## Pré-requisitos

- GCC ou MinGW configurado;
- Raylib instalada corretamente;
- VS Code com suporte a C/C++.

## Como executar

1. Entre na pasta VsGrafico.
2. Compile o arquivo principal com a configuração adequada para Raylib.
3. Execute o programa gerado.

## Destaque para recrutadores

Além da lógica do jogo, esta versão demonstra capacidade de transformar um projeto acadêmico em uma interface mais próxima de uma aplicação real.
